# XiaoHuaEr Voice Pack Credits

These are temporary local voice assets for XiaoHuaEr's 2.0.0 workflow events.

## Source

- Spoken text: original short Companion prompts written for XiaoHuaEr.
- Audio source: generated locally with macOS built-in text-to-speech using `/usr/bin/say`.
- Third-party samples: none.

## Generation

The clips were produced as AIFF files from local text-to-speech output and then
referenced by `voice-manifest.json` for event-based playback.

## Files

- `launch-welcome-01.aiff`
- `launch-welcome-02.aiff`
- `focus-start-01.aiff`
- `focus-end-01.aiff`
- `reminder-due-01.aiff`
- `translation-done-01.aiff`
